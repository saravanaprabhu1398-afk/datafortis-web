# DataFortis Website - Project Plan

## Information Gathered
The task is to create a landing page website for "DataFortis" - a data infrastructure company. The page needs 5 main sections:
1. **Hero Section** - Main headline, subtext, and CTA button
2. **The Problem** - 4 pain points for growing companies
3. **What We Do** - 4 service offerings
4. **Reference Architecture** - Technical diagram showing the data stack
5. **CTA Section** - Call to action for booking a strategy call

## Plan
1. Create project directory: `datafortis-website`
2. Create `index.html` with complete landing page structure
3. Create `styles.css` for modern, professional styling
4. Create `script.js` for any interactivity

## Design Decisions
- Modern, clean, professional design suitable for B2B data infrastructure
- Color scheme: Deep blue/navy (#0a2540) with accent colors for CTAs
- Typography: Clean sans-serif fonts (Inter or system fonts)
- Responsive design for all screen sizes
- Visual architecture diagram using CSS/flexbox layout
